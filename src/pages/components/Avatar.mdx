# Avatar

Avatars are used to represent a person or organization. They can be used in a variety of places, including navigation, lists, tables, cards, and more.

## Usage

<Demo
  component={(props) => <Avatar {...props} />}
  configurator={[
    {
      name: "src",
      type: "text",
      value: "/avatar.avif",
      defaultValue: "",
    },
    {
      name: "name",
      type: "text",
      value: "John Doe",
      defaultValue: "",
    },
  ]}
  template={(props) => `
    import { Avatar } from 'nullwind';

    export default function Demo() {
      return <Avatar ${props} />;
    }

`}/>

## Avatar Fallbacks

If there is an error loading the src of the avatar, there are 2 fallbacks:

- If there's a name prop, we use it to generate the initials.
- If there's no name prop, you can use a custom icon.

<Demo
  component={(props) => (
    <div class="flex gap-3">
      <Avatar name="John Doe" />
      <Avatar icon={<UserIcon class="h-5 w-5" />} />
    </div>
  )}
  template={(props) => `
    import { Avatar } from 'nullwind';
    import { UserIcon } from "nullstack-heroicons/24/solid";

    export default function Demo() {
      return (
        <>
          <Avatar src="/not-found.avif" name="John Doe" />
          <Avatar src="/not-found.avif" icon={<UserIcon class="h-5 w-5" />} />
        </>
      );
    }

`}/>
