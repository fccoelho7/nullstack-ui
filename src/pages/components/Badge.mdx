# Badge

Badges are used to highlight important information, such as the number of unread messages.

## Usage

<Demo
  component={(props) => <Badge {...props}>My cool badge</Badge>}
  configurator={[
    {
      name: "color",
      type: "select",
      options: ["primary", "secondary", "info", "success", "warning", "danger"],
      value: "primary",
      defaultValue: "primary"
    },
  ]}
  template={(props) => `
    import { Badge } from 'nullwind';

    export default function Demo() {
      return <Badge ${props}>My cool badge</Badge>;
    }

`}/>
