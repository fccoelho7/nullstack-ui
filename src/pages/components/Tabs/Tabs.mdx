# Tabs

Use these responsive tabs components to create a secondary navigational hierarchy for your website or toggle content inside a container

## Usage

<Demo
  component={Preview}
  template={() => `
    import { Tabs } from 'nullwind';

    export default function Demo() {
      return (
        <Tabs onchange={(idx) => (this.idx = idx)}>
          <Tabs.Tab title="Profile" active={this.idx === 0}>
            Tabs content 1
          </Tabs.Tab>
          <Tabs.Tab title="Notifications" active={this.idx === 1}>
            Tabs content 2
          </Tabs.Tab>
          <Tabs.Tab title="Settings" active={this.idx === 2}>
            Tabs content 3
          </Tabs.Tab>
        </Tabs>
      );
    }

`}/>
