# Title

Use Title component to render h1-h6 headings with Mantine theme styles. By default, Title has no margins and paddings. You can change headings font-size, font-weight and line-height properties in `theme.title`.

Set `h` prop to render specific element (h1-h6), default `h` is 1:

## Usage

<Demo
  component={(props) => (
    <>
      <Title h={1}>This is a H1 title!</Title>
      <Title h={2}>This is a H2 title!</Title>
      <Title h={3}>This is a H3 title!</Title>
      <Title h={4}>This is a H4 title!</Title>
      <Title h={5}>This is a H5 title!</Title>
      <Title h={6}>This is a H6 title!</Title>
    </>
  )}
  template={(props) => `
    import { Title } from 'nullwind';

    export default function Demo() {
      return (
        <>
          <Title h={1}>This is a H1 title!</Title>
          <Title h={2}>This is a H2 title!</Title>
          <Title h={3}>This is a H3 title!</Title>
          <Title h={4}>This is a H4 title!</Title>
          <Title h={5}>This is a H5 title!</Title>
          <Title h={6}>This is a H6 title!</Title>
        </>
      );
    }

`}/>
