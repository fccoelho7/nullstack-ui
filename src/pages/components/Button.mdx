# Button

A button is an interface element that makes it possible to perform an action. It must clearly and
objectively represent the effect of the action when clicked. Do not use buttons to navigate between
pages, for that use Links.

## Button as Link

You can use the Button component as a Link by passing the `href` prop.

<Demo
  component={(props) => <Button {...props}>Click here!</Button>}
  configurator={[
    {
      name: "title",
      type: "text",
      value: "Nullwind",
      defaultValue: ""
    },
    {
      name: "href",
      type: "url",
      value: "https://nullwind.dev/",
      defaultValue: ""
    },

    {
      name: "target",
      type: "select",
      options: ["_blank", "_self", "_parent", "_top"],
      value: "_blank",
      defaultValue: ""
    },

]}
template={(props) => `
import { Button } from 'nullwind';

    export default function Demo() {
      return <Button ${props}>Click here!</Button>;
    }

`}
/>
