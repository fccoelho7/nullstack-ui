# Tab

Use these responsive tabs components to create a secondary navigational hierarchy for your website or toggle content inside a container

## Usage

<Demo
  component={Preview}
  template={() => `
    import { Tab } from 'nullwind';

    export default function Demo() {
      return (
        <Tab onchange={(idx) => (this.idx = idx)}>
          <Tab.Item title="Profile" active={this.idx === 0}>
            Tab content 1
          </Tab.Item>
          <Tab.Item title="Notifications" active={this.idx === 1}>
            Tab content 2
          </Tab.Item>
          <Tab.Item title="Settings" active={this.idx === 2}>
            Tab content 3
          </Tab.Item>
        </Tab>
      );
    }

`}/>
