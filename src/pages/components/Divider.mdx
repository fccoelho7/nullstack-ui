# Divider

Divider component renders a horizontal line. You can change style properties in `theme.divider`.

Set `label` prop to render a specific label, default `label` is "More".

## Usage

<Demo
  component={(props) => <Divider class="w-full" {...props} />}
  configurator={[
    {
      name: "label",
      type: "text",
      value: "More",
      defaultValue: "More",
    },
    {
      name: "labelPosition",
      type: "select",
      options: ["left", "center", "right"],
      value: "center",
      defaultValue: "center"
    },
  ]}
  template={(props) => `
    import { Divider } from 'nullwind';

    export default function Demo() {
      return <Divider ${props} />;
    }

`}/>

## With custom label

<Demo
  component={() => (
    <Divider
      label="Read More"
      labelProps={{
        tag: 'a',
        class: 'text-primary-500',
        href: 'https://nullwind.dev'
      }}
    />
  )}
  template={(props) => `
    import { Divider } from 'nullwind';

    export default function Demo() {
      return (
        <Divider
          label="Read More"
          labelProps={{
            tag: 'a',
            class: 'text-primary-500',
            href: 'https://nullwind.dev'
          }}
        />
      );
    }

`}/>
