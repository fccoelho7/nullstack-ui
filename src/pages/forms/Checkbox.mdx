# Checkbox

The code below can be used to create a fieldset of checkbox elements inside a form. We also created a checkbox variant with extra description and one in a disabled form.

## Usage

<Demo
  component={(props) => <Checkbox {...props} />}
  configurator={[
    {
      name: "label",
      type: "text",
      value: "My Field",
      defaultValue: "",
    },
    {
      name: "helper",
      type: "text",
      value: "This is a helper message",
      defaultValue: "",
    },
    {
      name: "error",
      type: "text",
      value: "",
      defaultValue: "",
    },
    {
      name: "required",
      type: "boolean",
      value: true,
      defaultValue: false,
    },
    {
      name: "disabled",
      type: "boolean",
      value: false,
      defaultValue: false,
    },
  ]}
  template={(props) => `
    import { Checkbox } from 'nullwind';

    export default function Demo() {
      return <Checkbox ${props} bind={this.field} />;
    }

`}/>
