# Rating

Rating component is used to display rating stars.

## Usage

<Demo
  component={(props) => <Rating {...props} />}
  configurator={[
    {
      name: "averageRate",
      type: "number",
      value: 3,
      defaultValue: 0,
    },
    {
      name: "disabled",
      type: "boolean",
      value: false,
      defaultValue: false,
    },
  ]}
  template={(props) => `
    import { Rating } from 'nullwind';

    export default function Demo() {
      return <Rating ${props} bind={this.field} />;
    }

`}/>
